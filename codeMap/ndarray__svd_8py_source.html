5 def svd(T, a, b, chis=None, eps=0, print_errors=0,
6  return_rel_err=False, break_degenerate=False, degeneracy_eps=1e-6):
7  """ Reshapes the tensor T have indices a on one side and indices b
8  on the other, SVDs it as a matrix and reshapes the parts back to the
9  original form. a and b should be iterables of integers that number
10  the indices of T.
11 
12  The optional argument chis is a list of bond dimensions. The SVD is
13  truncated to one of these dimensions chi, meaning that only chi
14  largest singular values are kept. If chis is a single integer
15  (either within a singleton list or just as a bare integer) this
16  dimension is used. If no eps is given, the largest value in chis is
17  used. Otherwise the smallest chi in chis is used, such that the
18  relative error made in the truncation is smaller than eps.
19 
20  An exception to the above is degenerate singular values. By default
21  truncation is never done so that some singular values are included
22  while others of the same value are left out. If this is about to
23  happen chi is decreased so that none of the degenerate singular
24  values is included. This default behavior can be changed with the
25  keyword argument break_degenerate=True. The default threshold for
26  when singular values are considered degenerate is 1e-6. This can be
27  changed with the keyword argument degeneracy_eps.
28 
29  If print_errors > 0 truncation errors are printed, with more
30  information as print_errors increases through 1,2 and 3.
31 
32  By default the function returns the tuple (U,s,V), where in matrix
33  terms U.diag(s).V = T, where the equality is appromixate if there is
34  truncation. If return_rel_err = True a fourth value is returned,
35  which is the ratio sum_of_discarded_singular_values /
36  sum_of_all_singular_values.
37  """
